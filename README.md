# GMP Batch-Record Auditor Landing Page

A minimalist, high-converting landing page for the GMP Batch-Record Auditor SaaS.

## Features
- **Responsive Design**: Mobile-first approach.
- **Fast Performance**: Semantic HTML and optimized CSS (no heavy frameworks).
- **Dark Mode**: Premium, modern aesthetic.

## How to Deploy
1. **GitHub Pages**: 
   - Upload this `website` folder to a GitHub repository.
   - Go to Settings -> Pages.
   - Select the `main` branch and `website` folder (or root if you move files up).
   
2. **Netlify/Vercel**:
   - Drag and drop the `website` folder into the Netlify/Vercel dashboard.

## Customization
- **Email Link**: Update the "Book a Demo" link in `index.html` (currently `mailto:guillermo@example.com`).
- **Logos**: Add SVG logos to the "Proof" section.
